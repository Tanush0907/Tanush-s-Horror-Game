# Tanush's Horror Game

A first-person horror game built in **Unity**, featuring an **interaction system**, **movement system**, and immersive lighting.  
This is an ongoing project where I’m experimenting with creating atmospheric horror experiences.

---

## Features
- Basic **player movement system** using CharacterController  
- **Interaction system** for light switches, doors, and objects like flashlights  
- Early **lighting and ambiance setup**  
- Prototype **environment** and test scene  

---

## Planned Features
- Sound & music triggers  
- AI enemy / chase mechanics  
- Main menu and settings

## Built With
- **Unity 6.2**  
- **C#**
